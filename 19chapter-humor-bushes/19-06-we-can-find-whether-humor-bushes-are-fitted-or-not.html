The Humor Book

13. Evaluatory information

We can think about what the range of fitted humor bushes are for any pair of people—not just between us and our romantic partner and our students, but also between us and our:

  • corrections officer
  • food vendor
  • lifting buddy
  • professor we’re working for
  • adolescent child’s romantic partner
  • social media audience
  • medical specialist
  • homeless patron

(and so on)

In each of these relationships, there’s both a humor dress code for specific instances—“is this specific humor attempt fitted?”—as well as a humor dress code for humor in general—“is this overall amount of humor fitted?”

To find humor bush dress codes, we can take our humor outcomes one-by-one. For each one, we’d think about what degree of characterization would be fitted or unfitted.

For instance:

We’d ask “to what degree should we characterize our relationship with our doctor (or neighbor, lover, or so on)? Should we never/occasionally/often/always symbolize close ties?

Then we’d go through other humor outcomes, like acceptably criticize, reveal personal views, and so on. As we go through each humor outcome, we’ll arrive at an intuitive sense of what would be too much or too little.

Standards. The humor dress code is set by our need for efficiency and cohesion—which is true for individual moments of humor as well as for humor bushes. We can use this to think about the humor dress code for humor bushes, especially for groups.

We do this by asking “how cohesive and effective does a tour bus have to be?”—or a classroom, waiting room, or so on. This can help us estimate the upper and lower ranges for the dress code of humor bushes.

It also helps us think about how this changes overtime. The dress code for a humor bush will change when someone stops being our client, and we start becoming friends outside of any commercial relation—since our need for efficiency and cohesion changes.
