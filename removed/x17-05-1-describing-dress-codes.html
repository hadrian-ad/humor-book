Agreement on a dress code creates stability

We often set dress codes in subtle ways—like when we give a half-hearted smile instead of a full smile. But other times we have explicit disagreements about it. We may ask that someone “be serious”—a request that they raise their standard of fittedness.

When we arrive at a shared understanding of a situation's need for effectiveness and cohesion, we feel good on an emotion network—when we don't arrive at a shared understanding, things feel destructive:

Emma pulls out her phone, but is stupefied to see she accidentally grabbed Owen’s phone. Zoe doubles over with laughter at Emma’s expression.

Emma: (stupefied) “It’s not funny! Where’s my phone?”

Zoe laughs harder, and Emma gets more and more annoyed. (←Zoe and Emma disagree about the need for effectiveness and cohesion)

Emma: “I wish you would take this seriously.”

Zoe: *smiling* “Okay, okay, sorry.” (←Zoe and Emma are able to finalize a dress code)

- OR -

Emma: “I wish you would take this seriously.”

Zoe: *smiling* “You should see your face right now! Shocked Pikachu.” (←Zoe and Emma are unable to finalize a dress code)

Emma: “…”

When we agree with the wider, finalized dress code, we can comfortably join the emotion network and synchronize in a ways that feel acceptable to everyone, including ourselves.

When we don’t agree to the finalized dress code, we often feel uncomfortable and unintegrated—and we’ll leave as quickly if we can, and won’t want to return to these emotion networks.

The exact dress code is never precise—instead, a dress code functions with ambiguity, disagreements, conflicting motivations, inconsistent enforcement, incomplete consensus, and so on.

At first, it might seem dress codes function despite this ambiguity—but they actually function because of this ambiguity.

Consider:

Sometimes, we associate ambiguity with disorder or inefficiency. But in this case, the ambiguity gives us the flexibility needed for rich functionality. It allows for broad participation, consensus-seeking, exploration of boundaries, information-sharing, multiple ways of integrating, and so on.

Describing. We already have plenty of ways to describe dress codes in our day-to-day language—especially when we disagree with them. When we think the dress code is too high, we use words like buzzkill, joyless, snooty, stuck-up. When we think the dress code is too low, we use words like disrespectful, flippant, informal, permissive, unserious, and so on.

If a group seems snooty to us, the dress code is higher that we feel like agreeing to—and we won’t want to to fully join the emotion network. Of course, to the people there, it doesn’t feel snooty in a bad way—they’ve agreed to a high dress code for reasons that seem justified to them.

When we don’t agree with the dress code, we often either resist it or don’t know how to conform—and either way, we feel out of place.
