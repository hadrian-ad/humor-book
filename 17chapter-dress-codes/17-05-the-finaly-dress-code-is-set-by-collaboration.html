13. Evaluatory information

Each person estimates the dress code, but not everyone arrives at the same estimation. Usually, there’s some disagreement:

To finalize the dress code, we compromise toward each other:

The dress code is set collaboratively with who we’re around.

When we treat everyone around us as equals, we’ll set the dress code somewhere in the middle of where everyone thinks it should be.

If they’re not treating us as an equal, but instead low-ranking us, they’ll try to set the dress code unilaterally, without compromising. For instance, our boss’s boss may try to set the dress code without our input, even if the formality or informality seems strange to us—“Heyhey, I’m Steve!” (strangely informal) or “My name is Steven Alexander Charles III” (strangely formal).

Our reaction to this depends on how we feel being about being low-ranked. If the low-ranking seems justified or normal to us, then we’re simultaneously high-ranking them—and letting them set the dress code feels acceptable. But if they low-rank us and we don’t high-rank them, we’ll find obedience to their unilateral dress code distasteful or degrading.

Sometimes arriving at a shared dress code can be done precisely, like how close friends are acutely aware of the dress code around each other. Other times, it’s less precise, like how internet forums can have a broad gray zone between what’s considered fitted and unfitted.

Consider:

We do similar collaborations all the time. If we swear a lot and our friend doesn’t, we’ll compromise—we’ll swear less, and they’ll be okay with some more swearing. The exact compromise is weighted toward who cares most about it.

We often set dress codes in subtle ways. But sometimes we have explicit disagreements about it—like when we feel someone should “be serious” and raise their standard of fittedness.

Emma pulls out her phone, but is stupefied to see she accidentally grabbed Owen’s phone. Zoe doubles over with laughter at Emma’s expression.

Emma: (stupefied) “It’s not funny! Where’s my phone?”

Zoe laughs harder, and Emma gets more and more annoyed. (←Zoe and Emma disagree about the need for effectiveness and cohesion)

Emma: “I wish you would take this seriously.”

Zoe: *smiling* “Okay, okay, sorry.” (←Zoe and Emma are able to finalize a dress code)

Emma: “I wish you would take this seriously.”

It feels destructive when we can't arrive at a shared understanding of a situation’s need for effectiveness and cohesion.

When we agree with the wider, finalized dress code, we can comfortably join the emotion network and synchronize in a ways that feel acceptable to everyone, including ourselves.

When we don’t agree to the finalized dress code, we often feel uncomfortable and unintegrated—and we’ll leave as quickly if we can, and won’t want to return to these emotion networks.

The exact dress code is never precise—instead, a dress code functions with ambiguity, disagreements, conflicting motivations, inconsistent enforcement, incomplete consensus, and so on.

At first, it might seem dress codes function despite this ambiguity—but they actually function because of this ambiguity.

Consider:

Sometimes, we associate ambiguity with disorder or inefficiency. But in this case, the ambiguity gives us the flexibility needed for rich functionality. It allows for broad participation, consensus-seeking, exploration of boundaries, information-sharing, multiple ways of integrating, and so on.

Describing. We already have plenty of ways to describe dress codes in our day-to-day language—especially when we disagree with them. We use words like buzzkill, joyless, snooty, stuck-up, and so on when we think the dress code is too high. We use words like disrespectful, flippant, informal, permissive, unserious, and so on when we think the dress code is too low.

If a group seems snooty to us, the dress code is higher that we feel like agreeing to—and we won’t feel able to fully join the emotion network, and won’t really want to. Of course, to the people there, it doesn’t feel snooty in a bad way—they’ve agreed to a high dress code for reasons that seem justified to them.

When we don’t agree with the dress code, we often either resist it or don’t know how to conform—and either way, we feel out of place.
