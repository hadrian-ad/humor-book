The Humor Book

Higher dress codes ensure higher group coordination

When a group wants to accomplish more in less time, the humor dress code goes up. That's because the humor dress code is set by shared needs for efficiency and cohesion.

Efficiency

We’ve seen before how fittedness means a humor attempt is doing many things at once:

Justin’s walking out of the English department when Dave catches him.

Dave: *smiling* “Justin! The department’s going to get some drinks tonight. You should come!”

Justin: *shaking head* *smiling* "Thanks, but I’d better pass. The mountain of sobriety, you know, Tantalus and Sisyphus. Maybe another event!” (←reveals a personal trait, avoids conflict (while declining an invitation), deals with a taboo topic (alcoholism), supports an argument (that he has a valid excuse to decline), shares a perspective (he’s sober), and so on)

Dave: *laughs* “Sure, I get it. Maybe another time, another event then!”

Justin was efficient—he was doing many relevant things at once, very quickly.

Consider:

Simply saying "no" in a funny voice wouldn't have been more efficient, despite using fewer words. That's because "no" would just be doing one thing (declining), instead of doing many things (dealing with a taboo topic, supporting an argument, sharing a perspective, and so on).

By setting a higher dress code, we encourage efficient humor attempts, and discourage inefficient humor attempts. A higher dress code raises our standard of minimum effectiveness.

For instance:

If ten people take turns giving feedback on a proposal, we’ll want to move through everyone efficiently—and we know that if someone takes time to make an unfitted humor attempt (“totally re-markable!”), they’ll slow the process down without justification. To avoid this, we raise the dress code.

A higher standard of fittedness allows groups to accomplish more in less time, while weeding out less efficient things—but we only apply this higher standard when there’s actually something we want to accomplish efficiently.

Cohesive

Whenever it’s important or difficult to maintain cohesion, the dress code goes up. That's because the more fitted a humor attempt is, the more likely an emotion network will have synchronized evaluations of it.

We experience this all the time, like when we finally meet our close friend's partner for the first time:

Trevor: *sitting on the deck* “Hey. Do you think you could do it in a canoe without tipping it?” (←low need for cohesion with friends↔︎low dress code)

Minh: “Screw in a… canoe?” *laughs*

Trevor: *laughs* “Yeah—shagging no wagging!”

Minh: *laughs* “Dude, I could do it in ten canoes without tipping it.”

Yasmin comes out onto the deck, smiling. (←dress code increases)

Trevor: *beaming* “This is Yasmin! Yasmin, this is Minh”

Minh: *stands up* *smiling* *politely* “Nice to meet you! Trevor keeps saying wonderful things about you.” (←high need for cohesion when meeting new people↔︎higher dress code)

Yasmin: *smiling* “Thanks! Trevor says you’re a sculptor? That’s so neat!”

When we want everyone to stay synchronized with everyone else, we collectively avoid humor attempts that might desynchronize our emotion network.

Consider:

A conference involving many religious denominations will have a higher dress code than a conference of just one denomination.

That doesn’t mean there’ll be no humor. Instead, it just means the minimum fittedness goes up, since the conference wants to avoid desynchronizing the emotion network and hardening people into subnetworks. A higher dress code keeps the event cohesive.

Our humor dress code is the minimum standard for the complexity and coherence of our interactions—and this standard is constantly changing as the emotional situation changes.

Consider:

Advocacy organizations, charities, classrooms, international offices, big-tent political parties, public agencies, and so on all have high dress codes—since they’re goal-specific groups (need effectiveness) and have varied clients, customers, members, supporters, and so on (need cohesion).

These organizations want to avoid pointlessly fragmenting the emotional networks associated with them, while at the same time promoting efficient and effective communication. This means that our humor dress code goes up while actively participating in these groups, and goes down again when we’re not.

Relativity.

Sometimes we’re surprised that humor attempts are received differently in different places. We sometimes think there should be a flat dress code everywhere—so what's good among friends should be good in a classroom or a public social media profile.

But this gives a double standard for humor compared to other emotions. We're not surprised that attempts to make others feel affection, anger, exuberance, frustration, pleasure, sadness, and so on are treated differently in different emotional situations.

A sultry photo of ourselves or a long frustrated rant is is treated one way one place, and another way another place. And for the same reasons, a joke is treated one way in one place, and another way in another.

Since humor is a regular emotion, not surprising that humor attempts are treated differently in different emotional situations—since this is just like any emotional attempt.
