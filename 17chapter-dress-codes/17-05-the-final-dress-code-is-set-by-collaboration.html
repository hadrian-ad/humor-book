The final dress code is set by collaboration

We set the dress code collaboratively with everyone around us.

Humor dress codes are set by an agreed understanding of the need for efficiency and cohesion

Everyone around us estimates the need for efficiency and cohesion. But since not everyone arrives at the same estimation, we compromise toward each other to finalize the dress code:

When we treat everyone around us as equals, we’ll compromise by setting the dress code somewhere in the middle of where everyone thinks it should be.

Consider:

We arrive at agreements like this all the time. If we swear a lot and our friend doesn’t, we’ll compromise—we’ll swear less, and they’ll be okay with some more swearing. The exact compromise is weighted toward who cares most about it.

This doesn't happen instantaneously—it happens like a mutual game of warmer and colder. We adjust over time through mutual feeback until we eventually reach a point where both of us are comfortable.

When we agree with the wider, finalized dress code, we feel comfortable and integrated on the emotion network—and we'll feel able to synchronize to the network in ways that feel acceptable to everyone, including ourselves.

When we don’t agree to the finalized dress code, we often feel uncomfortable and unintegrated—and we won't feel able to synchronize to the network in ways that feel acceptable to everyone.

Precision. The exact dress code is never precise—instead, a dress code functions with ambiguity, disagreements, conflicting motivations, inconsistent enforcement, incomplete consensus, and so on.

At first, it might seem dress codes function despite this ambiguity—but they actually function because of this ambiguity.

Sometimes, we associate ambiguity with disorder or inefficiency. But in this case, the ambiguity gives us the flexibility needed for rich functionality. It allows for broad participation, consensus-seeking, exploration of boundaries, feedback, information-sharing, multiple ways of integrating, and so on.
