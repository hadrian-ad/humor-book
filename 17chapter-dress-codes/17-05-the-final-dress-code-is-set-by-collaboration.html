The final dress code is set by collaboration

We set the dress code collaboratively with everyone around us. Each person estimates the dress code—but not everyone arrives at the same estimation.

So to finalize the dress code, we compromise toward each other:

When we treat everyone around us as equals, we’ll compromise by setting the dress code somewhere in the middle of where everyone thinks it should be.

Consider:

We reach implicit agreements like this all the time. If we swear a lot and our friend doesn’t, we’ll compromise—we’ll swear less, and they’ll be okay with some more swearing. The exact compromise is weighted toward who cares most about it.

This happens like a mutual game of hot and cold—we adjust over time through mutual feeback. Eventually, we reach a point where both of us are comfortable.

When we agree with the wider, finalized dress code, we feel comfortable and integrated on the emotion network. We'll be able to synchronize to the network in ways that feel acceptable to everyone, including ourselves.

When we don’t agree to the finalized dress code, we often feel uncomfortable and unintegrated—and we’ll leave that emotion network as quickly if we can, and won’t want to return.

Consider:

The exact dress code is never precise—instead, a dress code functions with ambiguity, disagreements, conflicting motivations, inconsistent enforcement, incomplete consensus, and so on.

At first, it might seem dress codes function despite this ambiguity—but they actually function because of this ambiguity.

Sometimes, we associate ambiguity with disorder or inefficiency. But in this case, the ambiguity gives us the flexibility needed for rich functionality. It allows for broad participation, consensus-seeking, exploration of boundaries, feedback, information-sharing, multiple ways of integrating, and so on.

Describing. Dress codes aren't new to us—we already talk about them in our day-to-day language. This is especially obvious when we disagree with the humor dress code.

When we think the dress code is too high, we use words like buzzkill, joyless, snooty, stuck-up:

When we think the dress code is too low, we use words like disrespectful, flippant, informal, permissive, unserious, and so on.

We don't want to spend much time on an emotion network with a dress code we don't agree with—we don't like being somewhere that we'd describe as snooty:

Of course, to the people there, it doesn’t feel snooty in a bad way—they’ve agreed to a high dress code for reasons that seem justified to them. It's just a normal, comfortable dress code to them—the way any dress code we agree with seems to us.

When we don’t agree with the dress code, we often either resist it or don’t know how to conform to it;s—and either way, we feel out of place.
