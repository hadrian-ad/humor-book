13. Evaluatory information

To be fitted, a humor attempt needs a minimum amount of fittedness. This minimum amount of fittedness is what we’re calling the humor dress code:

The humor dress code changes situation by situation—like how the humor dress code is higher at the office and lower when we’re at home:

The dress code is constantly adjusting, not just hour by hour, but minute by minute—and even second by second.

      The dress code goes up when an all-staff meeting starts, and goes down when the meeting finishes
      The dress code goes down when we’re walking down the hall with a friend, and goes up when the fire alarm goes off
      The dress code goes down when we’re hanging out with our partner at home, and goes up when we start discussing a big financial decision

When a humor attempt is fitted, it’s seen as justified—it fits the situation. When the humor attempt is unfitted, it’s not seen as justified.
