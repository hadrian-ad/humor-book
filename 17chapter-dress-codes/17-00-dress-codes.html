17. Dress codes

We've been looking at the fittedness of humor or humor attempts—how well humor fits in a situation:

But to make the most of an idea like “fittedness,” we’ll also need what we’ll call a humor dress code—which is how much fittedness is required by a given situation.

We can understand this with an analogy to money. Let’s say we have $5. Is $5 enough, or not enough?

We can’t answer this question until we know what we want to buy! $5 is enough if we want to buy candy—it's not enough if we want to buy an entire pizza.

We’ll be doing the same thing with fittedness. If our humor attempt has “5” fittedness, is that enough (fitted), or not enough (unfitted)?

Just like we don't know if $5 is enough until we know what we want to use it for, we won’t know if "5" fittedness is enough until we know the humor dress code!

We’ll be going over this in greater detail throughout this chapter.

Fitted and unfitted. In this book, something that's "fitted" is above dress code, while something that's "unfitted" is below the dress code.

That means fittedness is relative to a dress code. A humor attempt with fittedness of "5" will be unfitted if the dress code is "6," but fitted if the dress code is "4":

That's why knowing fittedness by itself isn't enough to decide whether something is fitted or unfitted—since we also need the dress code.
