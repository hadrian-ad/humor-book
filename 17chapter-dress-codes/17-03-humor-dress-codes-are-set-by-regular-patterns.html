The Humor Book

Humor dress codes are set by regular patterns

At first glance, humor dress codes can seem arbitrary. But dress codes follow a regular pattern—they increases or decreases as we prioritize efficiency and cohesion:

Trevor, Yasmin, and some friends are going camping. They’re trash-talking each other as they drive, having a good time. (←there’s a low dress code while passing time↔︎little need for efficiency and cohesion)

Trevor: “Shut up everyone, the exit’s coming up. I need to hear the phone.” (←there’s a higher dress code while navigating↔︎greater need for efficiency and cohesion)

They arrive at the state park, and haul their bags out of the car. (←there’s a lower dress code doing menial tasks↔︎little need for efficiency and cohesion)

They work together to assemble the tents. Someone reads the instructions while others try to get the poles aligned. (←there’s a higher dress code when working together to accomplish something↔︎greater need for efficiency and cohesion)

Afterward they sit down to eat some food together at the picnic tables. (←there’s a lower dress code eating with friends↔︎little need for efficiency and cohesion)

Soon, other campers sit down at the picnic tables near them. (←there’s a higher dress code eating around strangers↔︎greater need for efficiency and cohesion)

We can see this pattern in any comparative situations. The dress code changes based on our need for efficiency and cohesion:

  • There’s a higher dress code when we’re at the register than when we’re at the back of the line
  • There’s a higher dress code during the meeting than when the meeting has finished
  • There’s a higher dress code when the teacher is explaining the assignment at the end of the period than when the teacher is welcoming the class at the beginning
  • There’s a higher dress code when we’re making plans with our friends to meet up later than when we’re with our friends and not making any plans
  • There’s a higher dress code when something serious is happening in the movie than when something serious isn’t happening in the movie

(and so on)

Consider:

This is obvious when we think about emergencies. The need for efficiency and cohesion goes up during emergencies, which means the humor dress code goes up too. This higher dress code then screens out things that don't help people be efficient or cohesive—like long knock, knock jokes.

But the higher dress code doesn’t mean humor or humor attempts go away, since during emergencies we'd still want to feel we’re surviving stressful situations, symbolizing close ties, supporting arguments, easing tension, and so on—and feel connection, insight, safety, and so on.

A higher dress code allows humor or humor attempts that promote efficiency and cohesion—while weeding out humor attempts that don't promote efficiency or cohesion.

Amount. Our first impression might be that the higher the dress code, the less humor there is. But that’s not true to our experiences. We can go get dinner with a friend at a casual diner (low dress code) and still have a dead conversation (little humor)—since a low dress code doesn’t compel lively conversation.

Similarly, if we dress up and go to an art gallery with our friend (high dress code), we can still smile and laugh (lots of humor)—a high dress code doesn’t compel frowning and boredom.

The dress code doesn't predict how much humor there'll be. A higher dress code just means we take more care in how we generate humor, which doesn’t always mean there’s less humor—it just means we take more care around humor.
