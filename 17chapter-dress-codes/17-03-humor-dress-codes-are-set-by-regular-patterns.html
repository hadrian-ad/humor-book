The Humor Book

Humor dress codes are set by regular patterns

At first glance, humor dress codes can seem arbitrary. But they follow a regular pattern—the dress code increases or decreases as we prioritize efficiency and cohesion.

For instance:

The need for efficiency and cohesion goes up during an emergency—and so the humor dress code goes up too. The higher dress code screens out things that don’t help people be efficient or cohesive—like unfitted knock, knock jokes.

But the higher dress code doesn’t mean humor or humor attempts go away, since we’d still want to feel we’re surviving stressful situations, symbolizing close ties, supporting arguments, easing tension, and so on—and feel connection, insight, safety, and so on.

These things are efficient and build cohesion—so a higher dress code allows these humor attempts, while weeding out less efficient or less cohesion-creating humor attempts.

The dress code shifts in response to the need for efficiency and cohesion:

Trevor, Yasmin, and some friends are going camping. They’re trash-talking each other as they drive, having a good time. (←there’s a low dress code while passing time↔︎little need for efficiency and cohesion)

Trevor: “Shut up everyone, the exit’s coming up. I need to hear the phone.” (←there’s a higher dress code while navigating↔︎greater need for efficiency and cohesion)

They arrive at the state park, and haul their bags out of the car. (←there’s a lower dress code doing menial tasks↔︎little need for efficiency and cohesion)

They work together to assemble the tents. Someone reads the instructions while others try to get the poles aligned. (←there’s a higher dress code as working together to accomplish something↔︎greater need for efficiency and cohesion)

Afterward they sit down to eat some food together at the picnic tables. (←there’s a lower dress code eating with friends↔︎little need for efficiency and cohesion)

Soon, other campers sit down at the picnic tables near them. (←there’s a higher dress code eating around strangers↔︎greater need for efficiency and cohesion)

We can see this pattern in any comparative situations. The dress code changes based on our need for efficiency and cohesion:

  • There’s a higher dress code when we’re at the register than when we’re at the back of the line
  • There’s a higher dress code during the meeting than when the meeting has finished
  • There’s a higher dress code when the teacher is explaining the assignment at the end of the period than when the teacher is welcoming the class at the beginning
  • There’s a higher dress code when we’re making plans with our friends to meet up later than when we’re with our friends and not making any plans
  • There’s a higher dress code when something serious is happening in the movie than when something serious isn’t happening in the movie

(and so on)

Amount. Our first impression might be that the higher the dress code, the less humor there is. But that’s not true to our experiences—like how we can go get dinner with a friend and still have a dead conversation. The low dress code doesn’t compel lively conversation.

Similarly, a high dress code doesn’t somehow suck humor from an experience. If we dress up and go to a formal gallery opening with our friend, we can still smile and laugh—the high dress code doesn’t compel frowning and boredom.

The dress code doesn't predict how much humor there'll be. A higher dress code means we take more care in how we generate humor, which doesn’t always mean there’s less humor—it just means we take extra care.
